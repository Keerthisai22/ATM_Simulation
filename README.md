# ATM Simulation System (Python)

A simple Python program that simulates basic ATM operations using functions and loops.

## Features
- Deposit money
- Withdraw money with balance check
- View current balance
- Menu-driven system
- Continuous operation until exit

## Technologies Used
- Python 3

## How to Run
1. Download or clone the repository
2. Run the program:
   python atm_simulation.py
